# 2048 Game

A classic 2048 game implementation using HTML, CSS, and JavaScript.

## How to Play

1. Use arrow keys to move tiles
2. When two tiles with the same number touch, they merge into one
3. Add numbers to reach 2048!

## Features

- Clean and modern UI
- Responsive design
- Score tracking
- New game button

## Technologies Used

- HTML5
- CSS3
- JavaScript
